Login Template Steps

  1. Reuse the same login validation rules from earlier lessons before checking credentials.
  2. Call findUser(username) after the form is already valid.
  3. If no user exists, show User not found. on the username field.
  4. If the password does not match, show Incorrect password. on the password field.
  5. If both match, call saveSession(user) and redirect to index.html.
